ファクタリングを利用した個人事業主が最も悩むのが、**確定申告での経費処理と仕訳方法**です。「手数料はどの勘定科目で処理すればいいのか」「消費税の扱いはどうなるのか」といった疑問は、多くのフリーランスが抱えています。

**ファクタリング手数料は「売上債権売却損」として経費計上が可能**です。この記事では、具体的な仕訳例から確定申告書への記載方法まで、ステップごとに解説します。

この記事でわかること：

- ファクタリング手数料の**正しい勘定科目**
- 2社間・3社間それぞれの**具体的な仕訳例**
- 確定申告書への**記載方法と注意点**
- **青色申告**での処理のポイント
- 税務調査で**指摘されないための注意点**

## ファクタリング手数料の勘定科目

ファクタリング手数料を経費処理する際、使用する勘定科目は以下のいずれかです。

| 勘定科目 | 使用場面 | 備考 |
|----------|----------|------|
| **売上債権売却損** | 最も正確な勘定科目 | 会計ソフトにない場合あり |
| **支払手数料** | 一般的な代替科目 | 多くの会計ソフトに標準搭載 |
| **雑損失** | 金額が小さい場合 | 頻繁な利用には不向き |

**推奨は「売上債権売却損」**です。ファクタリングの本質は売掛金の売却であり、手数料部分は売却損に該当します。

ただし、個人事業主向けの会計ソフト（freee・マネーフォワード等）では「売上債権売却損」が標準で用意されていないこともあります。その場合は**「支払手数料」で代替しても問題ありません**。

**重要なのは、一度決めた勘定科目を毎回統一して使うこと**です。途中で変更すると税務調査で質問される原因になります。

## 2社間ファクタリングの仕訳例

2社間ファクタリングでは、売掛金を業者に売却し、後日売掛先から入金された代金を業者に支払います。

### ステップ1：売掛金の発生

```
（借方）売掛金 500,000円 ／ （貸方）売上 500,000円
```

### ステップ2：ファクタリング契約・入金時

手数料10%（50,000円）の場合：

```
（借方）普通預金 450,000円 ／ （貸方）売掛金 500,000円
（借方）売上債権売却損 50,000円
```

### ステップ3：売掛先から入金、業者へ支払い

```
（借方）普通預金 500,000円 ／ （貸方）預り金 500,000円
（借方）預り金 500,000円 ／ （貸方）普通預金 500,000円
```

## 3社間ファクタリングの仕訳例

3社間ファクタリングでは、売掛先が直接ファクタリング業者に支払うため、仕訳がシンプルです。

### ステップ1：売掛金の発生

```
（借方）売掛金 500,000円 ／ （貸方）売上 500,000円
```

### ステップ2：ファクタリング契約・入金時

手数料5%（25,000円）の場合：

```
（借方）普通預金 475,000円 ／ （貸方）売掛金 500,000円
（借方）売上債権売却損 25,000円
```

3社間の場合は売掛先からの入金回収が不要なため、**ステップ2で仕訳が完了**します。

## 消費税の取り扱い

ファクタリング手数料の消費税の扱いは重要なポイントです。

**ファクタリング手数料は「非課税取引」**に該当します。金銭債権の譲渡は消費税法上の非課税項目に含まれるためです。

| 項目 | 消費税区分 |
|------|:---:|
| ファクタリング手数料 | **非課税** |
| 事務手数料・登記費用 | **課税** |
| 売掛金の元本部分 | **対象外** |

注意点として、ファクタリング手数料とは別に**事務手数料や登記費用が発生する場合は課税取引**です。請求明細をしっかり確認し、区分を正しく処理しましょう。

## 確定申告書への記載方法

### 青色申告の場合

青色申告決算書の**損益計算書**に記載します。

1. 「売上債権売却損」は**「その他の経費」欄**に記載
2. 「支払手数料」で処理した場合は**該当欄**に記載
3. 青色申告特別控除（65万円または10万円）は**問題なく適用可能**

### 白色申告の場合

収支内訳書の**経費欄**に記載します。

- 「支払手数料」または「雑費」として記載するのが一般的
- 金額が大きい場合は**内訳を別途メモ**しておくと安心

**青色申告を選択している場合、ファクタリング手数料を経費計上しつつ65万円控除も受けられる**ため、節税効果が高くなります。

## 税務調査で指摘されないための3つのポイント

ファクタリングの経費処理で税務調査を受けた際に問題とならないよう、以下の3点を徹底しましょう。

### ポイント1：契約書を必ず保管する

ファクタリング業者との**契約書・取引明細書は7年間保管**が必要です。オンライン完結型の場合も、PDFをダウンロードして保存しておきましょう。

### ポイント2：勘定科目を統一する

前述のとおり、**使用する勘定科目は毎回統一**してください。年度ごとに変えたり、取引ごとに変えたりすると、税務署から確認を受ける可能性があります。

### ポイント3：取引の実態を説明できるようにする

ファクタリングの**取引経緯・必要性を説明できる資料**を用意しておくと安心です。売掛金の入金サイトが長い、急な支払いが発生した等の理由を記録しておきましょう。

## 会計ソフトでの処理方法

主要な会計ソフトでのファクタリング処理方法を紹介します。

| 会計ソフト | 勘定科目の追加 | 処理のしやすさ |
|-----------|:---:|:---:|
| **freee** | カスタム科目で追加可能 | 簡単 |
| **マネーフォワード** | 勘定科目設定から追加 | 簡単 |
| **弥生会計** | 勘定科目登録から追加 | やや手間 |

いずれのソフトでも、**「売上債権売却損」をカスタム勘定科目として追加登録**すれば、正確な処理が可能です。

## まとめ

ファクタリングと確定申告について、押さえるべきポイントをまとめます。

- ファクタリング手数料の勘定科目は**「売上債権売却損」が最も正確**
- 会計ソフトにない場合は**「支払手数料」で代替可能**
- ファクタリング手数料は**消費税非課税**、事務手数料は課税
- 2社間と3社間で**仕訳のステップ数が異なる**ため注意
- **勘定科目の統一・契約書の保管**が税務調査対策の基本
- 不安な場合は**税理士に相談**し、正確な処理を心がけよう
